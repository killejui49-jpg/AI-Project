# L'Intelligence Artificielle - Projet Scolaire

**Presentation interactive sur l'Intelligence Artificielle**

- **Etudiant** : Ayoub Assouar
- **Niveau** : Tronc Commun Scientifique
- **Matiere** : Informatique

## Contenu de la Presentation

1. Qu'est-ce que l'IA ?
2. Histoire de l'IA
3. Les types d'IA (Faible, Forte, Super IA)
4. Comment fonctionne l'IA ? (Donnees, Algorithmes, Machine Learning, Reseaux de Neurones)
5. Applications de l'IA (Medecine, Transport, Education, Jeux, Finance, Environnement)
6. L'IA dans la vie quotidienne
7. Avantages et Inconvenients
8. L'avenir de l'IA
9. Conclusion

## Comment Voir la Presentation

Ouvrez `index.html` dans un navigateur web, ou visitez la version en ligne via GitHub Pages.

**Navigation** : Utilisez les fleches du clavier (gauche/droite) ou les boutons en bas de l'ecran.
